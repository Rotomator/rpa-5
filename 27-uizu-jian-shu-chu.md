# UI组件-输出

## 输出方法

* 文本：默认为输出文本。常见的情况下输出为文本，内容准确并且速度非常快。所有的过程都是在后台进行的。
* 原生： 原生能够提取出文本数据，例如屏幕坐标的每一个字符。
* OCR - 众所周知的OCR的准确率并达不到100%，但是面对图片中文字，OCR能够给出有一定准确率的输出。

以下列出输出方法比对：

|  | 速度 | 准确 | 后台运行 | 文本位置 | 隐藏文本 | 支持Citrix |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| 文本 | 非常快 | 100% | 是 | 否 | 是 | 否 |
| 原生 | 快 | 100% | 否 | 是 | 否 | 否 |
| OCR | 慢 | 98% | 否  | 是 | 否 | 是 |
